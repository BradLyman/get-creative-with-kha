An exploration of random number generation techniques in haxe. Visual
inspection is a poor model, it may be worth doing this exploration with other
tools.

## How To Build

### Web Target

```
> node ../../kha/make html5
```

### Native Windows Target

```
> node ../../kha/make windows --compile --run
```
