A short demonstration for getting Kha initialized and graphics on the screen.
As a nice bonus, the performance is stellar.

## How To Build

### Web Target

```
> node ../../kha/make html5
```

### Native Windows Target

```
> node ../../kha/make windows --compile --run
```
